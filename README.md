# we4water initiatives official website

Official site for We4Water, a community initiative focused on improving water access, stewardship, and awareness.

## Overview
- Public-facing site for We4Water initiatives, news, and calls to action.
- Built as a static Next.js app and deployed via GitHub Pages from the `main` branch.

## Tech stack
- Next.js (App Router)
- React + TypeScript
- Global styling via CSS

## Getting started
1) Install dependencies: `npm install`
2) Run locally: `npm run dev` and open http://localhost:3000
3) Lint (optional): `npm run lint`

## Build & deployment
- Production build: `npm run build`
- Deployment: published to GitHub Pages from the `main` branch. Pushes to `main` trigger the build-and-publish workflow defined in repo settings.
